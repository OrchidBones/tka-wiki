挑战赛

挑战赛，又称复活赛，是由荣耀职业联盟对想注册成为正式职业队的普通玩家队伍设立的一项选拔赛事。每场比赛一般会在周五晚上举行。

挑战赛报名从八月开始，只要是满六人的团队，并且交了报名费，就有资格报名参加荣耀职业联盟举办的挑战赛，期限为1个月。

选拔赛是和联赛同步进行的，联赛常规赛结束的时候，选拔赛也会出现最终结果，下赛季的20支参赛队，就是在这个时候确定的。选拔赛现在每年报名的队伍都很多，继续发展下去，可能会以此为根基建立次级联赛。

挑战赛报名审核只需一个身份，中前期的比赛全都是在线上。后期才会转为线下。

线上赛事，是需要借用网游中的竞技场来完成。游戏方也会为此专门辟出一块挑战赛专用的竞技场。每一期的赛事房间都会完美地分布好，然后各队都会在赛程安排中找到自己比赛的房间号，如期到房间和对手进行比赛，系统自然会把结果记录下来。主客场用抛硬币来决定。

挑战赛是淘汰制，一场比赛打输了，就要出局。

挑战赛中途允许随意增减队员。

有关退役选手的规定中，完全没有提到挑战赛的事。不过从现有规则解读的话，退役选手参与挑战赛不存在违规。

夺得挑战赛冠军的队伍会获得联盟的支持，成为下一赛季拥有注册资格的正式职业队。即使该支队伍的部分条件无法满足，联盟也会想办法解决。
